# 大创管理系统简介 java354

## 项目概述

大创管理系统是一个服务于大学生创新创业项目的管理平台，旨在提供从项目申报到结项的全流程管理功能。系统主要包括院系管理员、指导老师和学生三种角色，通过高效的项目管理工具，促进项目管理工作的规范化和高效化。

## 技术栈

- 后端：Spring Boot
- 前端：Vue 2 + ElementUI
- Node.js版本：14.x
- 运行环境：JDK 1.8 + MySQL 5.7及以上
